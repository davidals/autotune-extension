How to Use Autotune

1. Getting Started

First, you'll need an OpenAI API key. You can get one from the OpenAI platform.

  1. Sign up for an OpenAI account if you haven't already
  2. Navigate to the API keys section
  3. Create a new API key
  4. Copy and paste it into the API Key field above

2. Using the Extension

Autotune can be used on any text input field across the web:

  1. Click the Autotune extension icon in your browser toolbar
  2. Select your desired model and adjust parameters
  3. Paste or type your text in the input field
  4. Click "Enhance" to process your text
  5. Review the enhanced text and click "Accept" or "Revert"

3. Understanding Parameters

Fine-tune your text enhancement with these parameters:

  • Verbosity: Control the length and detail of the output
  • Formality: Adjust the level of formality in the text
  • Tone: Set the emotional tone of the message
  • Complexity: Modify the complexity of the language
  • Persuasiveness: Adjust the persuasive power of the text

4. Tips for Best Results

  • Start with a clear, well-structured input text
  • Experiment with different parameter combinations
  • Use the preview feature to see changes before accepting
  • Save your preferred parameter settings for quick access

5. Troubleshooting

If you encounter any issues:

  • Verify your API key is valid and has sufficient credits
  • Check your internet connection
  • Try refreshing the page if the extension isn't responding
  • Contact support if issues persist
